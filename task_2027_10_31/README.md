## 💬 Типы чатов

### 1. Общие чаты
**Описание:** Публичные чаты для обсуждения рынков
- **Доступность:** Все пользователи
- **Модерация:** Автоматическая + ручная
- **Темы:** Обсуждение рынков, новости, стратегии
- **Лимиты:** 1000 участников на чат

**Функции:**
- Отправка текстовых сообщений
- Отправка изображений и файлов
- Реакции на сообщения
- Цитирование сообщений
- Поиск по истории

### 2. Приватные чаты
**Описание:** Личные сообщения между пользователями
- **Доступность:** Только участники
- **Модерация:** Автоматическая
- **Приватность:** Полная конфиденциальность
- **Лимиты:** 1 на 1 общение

**Функции:**
- Личные сообщения
- Обмен файлами
- Голосовые сообщения
- Видеозвонки
- История сообщений

### 3. Групповые чаты
**Описание:** Чаты для групп пользователей
- **Доступность:** Участники группы
- **Модерация:** Автоматическая + модераторы группы
- **Темы:** Специализированные обсуждения
- **Лимиты:** 100 участников на группу

**Функции:**
- Групповые сообщения
- Управление участниками
- Настройки группы
- Файлообмен
- Календарь событий

### 4. Рыночные чаты
**Описание:** Чаты для обсуждения конкретных рынков
- **Доступность:** Участники рынка
- **Модерация:** Автоматическая + модераторы рынка
- **Темы:** Обсуждение конкретного рынка
- **Лимиты:** 500 участников на рынок

**Функции:**
- Обсуждение рынка
- Обмен мнениями
- Анализ и прогнозы
- Уведомления о событиях
- Интеграция с торговым интерфейсом

## 🔔 Система уведомлений

### 1. Push-уведомления
**Типы уведомлений:**
- **Торговые:** Исполнение ордеров, изменения цен
- **Рыночные:** Новые рынки, разрешение рынков
- **Социальные:** Новые сообщения, упоминания
- **Системные:** Обновления, технические уведомления

**Настройки:**
- Выбор типов уведомлений
- Время получения уведомлений
- Частота уведомлений
- Отключение уведомлений

### 2. Email-уведомления
**Типы писем:**
- **Еженедельные дайджесты:** Сводка активности
- **Важные события:** Критические уведомления
- **Маркетинговые:** Новые функции, акции
- **Безопасность:** Уведомления о безопасности

**Настройки:**
- Частота получения
- Типы писем
- Формат писем
- Отписка от рассылок

### 3. In-app уведомления
**Типы уведомлений:**
- **Всплывающие:** Важные уведомления
- **Баннеры:** Информационные сообщения
- **Бейджи:** Счетчики непрочитанных
- **Звуковые:** Звуковые сигналы

**Настройки:**
- Включение/отключение звуков
- Выбор типов уведомлений
- Настройка времени показа
- Персонализация
