### Шаг 3: Создание диаграммы Ганта в Excel

#### 3.1 Подготовка данных:
1. Открыть Excel и создать новый лист с названием "TASK_2025_10_31"
2. Заполнить таблицу данными из анализа README.md
3. Добавить столбцы для расчета дат окончания

#### 3.2 Создание диаграммы:
1. Выделить данные таблицы (столбцы A, C, D)
2. Перейти на вкладку "Вставка" → "Диаграммы" → "Линейчатая" → "Линейчатая с накоплением"
3. Настроить диаграмму:
   - Удалить синие сегменты (даты начала)
   - Оставить только оранжевые сегменты (продолжительность)
   - Настроить оси:
     - Вертикальная ось: обратный порядок категорий
     - Горизонтальная ось: формат дат

#### 3.3 Форматирование диаграммы:
1. **Заголовок:** "Диаграмма Ганта - MVP разработка (8-31 октября 2025)"
2. **Цвета для разных ролей:**
   - Blockchain Engineer: синий

   - Backend Engineer: красный

3. **Легенда:** показать соответствие цветов ролям
