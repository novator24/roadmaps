# Roadmap Analysis Project

## 📋 Описание

Проект для анализа .md файлов и генерации Excel файлов с диаграммами Ганта для платформы предсказательных рынков.

## 🚀 Быстрый старт

### Автоматическая настройка
```bash
# Запуск скрипта настройки
./setup.sh
```

### Ручная настройка
```bash
# Создание виртуального окружения
python3 -m venv venv

# Активация окружения
source venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt
```

## 🔧 Использование

```bash
# Активация окружения
source venv/bin/activate

# Запуск анализа
python solution.py

# Деактивация окружения
deactivate
```

## 📁 Структура проекта

```
roadmaps/
├── README.md                    # Основной roadmap
├── task_2025_10_31/README.md     # MVP задачи
├── task_2025_11_30/README.md     # Платежи
├── task_2025_12_31/README.md     # Безопасность
├── task_2026_10_31/README.md     # Аудит
├── task_2027_10_31/README.md     # Коммуникация
├── solution.py                  # Основной скрипт
├── requirements.txt             # Зависимости Python
├── setup.sh                    # Скрипт настройки
├── .gitignore                  # Исключения для Git
└── roadmap_analysis.xlsx        # Результат (генерируется)
```

## 📊 Результат

Скрипт создает Excel файл `roadmap_analysis.xlsx` с:

- **6 листов** с информацией о каждом этапе проекта
- **12 диаграмм Ганта** (по 2 на каждый лист)
- **Context7 анализ** сложности и зависимостей задач
- **Цветовое кодирование** приоритетов

## 🛠 Зависимости

- **openpyxl** - работа с Excel файлами
- **markdown** - парсинг Markdown файлов
- **beautifulsoup4** - парсинг HTML
- **typing-extensions** - поддержка типов

## 📝 Лицензия

Проект создан для анализа roadmap платформы предсказательных рынков.
